## SPECIAL COMBINATIONS & BONUSES

### Unlock Combinations
- Text Adventure + Fantasy = Unlocks RPG type
- Arcade + Table Tennis (named "Ping") = Unlocks Office topic
- Rhythm + Music Creation/Dance = Unlocks Motion-Control type
- Creating 3 Arcade games = Unlocks Racing type
- Creating 2 Board Games = Unlocks Strategy type (also unlocked in 1992)
- Creating 3 Adventure games = Unlocks Dinosaurs topic

### Rating Modifiers
- Sports Type + Sports Topic = +10% Gameplay bonus
- Online Game (1994-2000) = -5% all ratings
- MMORPG (before 2004) = -10% all ratings

### Year-Based Unlocks
- 1985: Board Game type unlocked automatically
- 1992: Strategy type available (if not already unlocked)
- 1994: Online Game type available (until 2000)
- 1996: MMORPG available (if RPG unlocked)
- 2000: Retro type unlocked

### Condition-Based Unlocks
- <40% hygiene (1978-1983) = Bugs topic
- Don't use desktop for 1 year = Idle type
- 90% happiness = Educational type
